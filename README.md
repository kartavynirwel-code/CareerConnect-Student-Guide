🚀 CareerConnect

A Full Stack Student Career & Education Guidance Web Application built using React, Spring Boot, MySQL and AWS.

📌 About The Project

CareerConnect is a web platform that helps students:

Explore career options based on stream

Search and discover colleges

View career details and growth paths

Manage their student account

Access structured educational guidance

This project demonstrates full-stack development skills and cloud deployment readiness.
